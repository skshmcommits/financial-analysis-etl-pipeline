# Project Specification  
## Automated Financial Research Data Pipeline
### Project Goal
Build an **end-to-end Python system** that:
1. Extracts financial market data from APIs
2. Cleans and validates the data
3. Stores the processed data in a SQL database
4. Computes financial analytics
5. Generates automated research reports
6. Provides a **CLI tool** to run the pipeline
The project should simulate **real-world financial research infrastructure used in banks and quant teams**.
---
